      • 学校简介company profil
      • 华商动力始创于1995年中国首都北京海淀区中关村，是中国最大的涉外知识产权专业代理机构，全国核心城市拥有31家直属分支机构，全面覆盖华南、华东、华中、华北、东北、西部及海外区域。 2009年05月04日华商动力集团的“黄埔军校”前身——华商动力旗下全资公司陕西汇泽知识产权代理有限公司成功迁址，为集团的再次突破、高速发展奠定了人才基础。

               2013年02月21日华商动力集团的总部——西安运营服务中心正式建成运行，为集团整体的发展提供更强大的后勤保障。为各分公司的技术支持提供强大后盾。目前，华商动力已为包括世界500强在内的100多万家客户提供了服务，与国家知识产权局、国家商标局、国家版权局、中华商标协会、香港知识产权署、国际保护工业产权协会（AIPPI）、国际商标协会（INTA）建立了战略合作关系。 2013年05月上旬，华商动力（北京）科技有限公司研发的新品“小马”正式上线，从此，华商动力借移动互联网开启了科技新篇章，实现了产业升级。

                2013年08月07日，华商动力学院正式挂牌成立，至此华商动力集团“黄埔军校”正式诞生，这将为华商动力集团传播企业文化、培养精英人才、规划发展战略提供了坚实的人才基础和理论储备。 华商动力（北京）科技有限公司是华商动力集团旗下全资子公司，是集团为实现战略产业升级全力打造的又一品牌。集团由此全力进军品牌推广产业，借助互联网开启了科技的新篇章。
